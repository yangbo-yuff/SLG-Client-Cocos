## 连接服务端

服务端部署成功后，修改客户端 GameConfig.ts 文件中的连接地址

```typescript
import { _decorator } from 'cc';
const GameConfig = {
    serverUrl: "ws://127.0.0.1:8004", //httpserver 地址
    webUrl: "http://127.0.0.1:8088",  //gateserver 地址
}
export { GameConfig };
```
